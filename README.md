# 🦠 COVID-19 Data Analysis (SQL Project)

## 📌 Overview

This project focuses on analyzing COVID-19 data using SQL to understand infection trends, death rates, and vaccination progress across different countries and continents.

The analysis was performed using real-world datasets and includes data cleaning, aggregation, and advanced SQL techniques.

---

## 🎯 Objectives

* Analyze total cases vs total deaths to understand fatality rates
* Identify countries with the highest infection rates relative to population
* Compare COVID-19 impact across continents
* Track vaccination progress over time
* Calculate percentage of population vaccinated

---

## 🛠️ Tools & Skills Used

* SQL (Joins, Aggregations, Window Functions)
* CTEs (Common Table Expressions)
* Temporary Tables
* Data Cleaning & Type Conversion
* Creating Views for analysis

---

## 📊 Key Analysis & Insights

### 🔹 Infection Rate Analysis

* Calculated percentage of population infected
* Identified countries with highest infection spread

### 🔹 Death Rate Analysis

* Computed death percentage (total deaths / total cases)
* Observed variation across countries

### 🔹 Continent-wise Impact

* Compared total deaths across continents
* Highlighted regional differences

### 🔹 Vaccination Trends (Advanced)

* Used window functions to calculate cumulative vaccinations
* Tracked vaccination progress over time

### 🔹 Population vs Vaccination

* Calculated percentage of population vaccinated
* Provided more meaningful insights than raw numbers

---

## 🔍 Key SQL Concepts Used

* `JOIN` – combining datasets
* `GROUP BY` – aggregation
* `SUM(), COUNT(), MAX()` – analysis
* `WINDOW FUNCTIONS` – rolling calculations
* `CTE` – simplifying complex queries
* `TEMP TABLES` – intermediate storage
* `VIEWS` – reusable queries

---

## 📁 Dataset

* CovidDeaths

## 💡 Key Takeaways

* Relative metrics (percentages) provide better insights than absolute values
* Window functions are powerful for time-based trend analysis
* Data cleaning is essential when working with real-world datasets
